2. Design Token Unification

Hardcoded colors (e.g., bg-white, text-gray-500) were replaced with semantic tokens defined in index.css.

Old Value New Semantic Token Purpose
bg-white bg-background Main background color, supports theming.
text-gray-900 text-foreground Primary text color.
text-gray-500 text-muted-foreground Secondary/muted text color.
border-gray-200 border-border Default border color.
bg-blue-600 bg-primary Primary action background.
